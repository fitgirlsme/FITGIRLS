# FITGIRLS SEO Master Brief

## 1. Overview & Business Goals
FITGIRLS & INAFIT is a premium body profile and fashion editorial photography studio based in Sinsa-dong, Gangnam, Seoul, Korea. 
The studio targets both local Korean clients and global clients (specifically from Japan, English-speaking countries, and Greater China) who want high-quality, professional body profile photography with styling and multilingual support.

### Primary SEO Goals:
- **Increase Organic Visibility**: Ranking high on Google (Global/Korea) and Naver (Korea) for primary and secondary keywords.
- **Multilingual Search Discovery**: Enable search engines to index and rank localized versions of the site (Korean, English, Japanese, Chinese).
- **Convert Traffic to Bookings**: Direct users smoothly from discovery (Archive/Gallery/Studios) to the reservation system.

## 3. SEO Constraints & Guidelines
1. **Dynamic Meta Tag Injection**: Since the site is a client-side rendered Single Page Application (SPA), metadata (title, description, OG tags) must be injected dynamically or managed through pre-rendering to allow modern search crawlers to scan correctly.
2. **Localizations**: Localized subpaths or query parameters must be properly indexed using standard `link rel="alternate" hreflang="..."` tags.
3. **Structured Data**: Consistent schema markup (`LocalBusiness`, `ImageObject`, `FAQPage`) must be served to search bots.
4. **Media Optimization**: Since it is a photography studio website with many large images, all photos must use optimized dimensions and formats (WebP/AVIF), with appropriate file names and descriptive ALT tags.
